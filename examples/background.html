Lazy Load

基于 jQuery 的图片懒加载插件。

Star Fork Download

设置为背景图和淡入效果

使用 CSS 设置容器背景，并开启淡入效果。注意这种方式图片载入完成以后会被设置为容器的背景图片。

<div class="lazy" data-original="img/bmw_m1_hood.jpg" style="width: 765px; height: 574px;"></div>
.lazy {
 margin: 10px auto;
 background: url(img/loading.gif) no-repeat center center;
}
$('div.lazy').lazyload({
  effect : 'fadeIn'
});
