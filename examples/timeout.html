Lazy Load

基于 jQuery 的图片懒加载插件。

Star Fork Download

设置加载时机

下面的示例中，页面加载完成 5 秒以后触发 sporty 事件时才会执行图片懒加载。

$(function() {
  $('img.lazy').lazyload({
    event : 'sporty'
  });
});

$(window).bind('load', function() {
  var timeout = setTimeout(function() {
    $('img.lazy').trigger('sporty');
  }, 5000);
});
